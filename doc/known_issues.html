Known Issues This section lists the significant problems we are aware of with this DDE release. All of them are on our "do list".

This version is an early release in the new Electron framework. It is made available only for testing purposes. It is not well tested. In general, users should not use it. Stick with the previous Chrome Apps version a while longer. Sorry.

  • Sending commands to hardware robot broken (but similation ok)
  • Speech recognition busted.
  • None of the apps for the 3 platforms are "signed", thus you get warnings when installing them about being "untrusted".
  • Linux distribution has some bad behavior with the menus that probably precludes effective use of DDE on Linux right now.
  • Dexter.draw_dxf needs extension to describe the drawing plane.
  • Dexter.run_gcode needs many extensions.
  • Some instructions lacking sections in the Reference Manual.
  • Train Dexter doesn't output a Job definition.
  • Documentation on ROS functionality
  • Support for ROS action clients needed.
